So, what exactly is React?

React is an open-source and client-side JavaScript library which is maintained by Facebook and is used for building complex single page applications. If you’re familiar with web technologies such as: - HTML5, JavaScript then react can be further understandable as HTML-in-JavaScript syntax popularly known as JSX. The most important feature of React is one can create their own custom elements like HTML elements which are reusable to quickly and efficiently build user interfaces. React makes it painless to create interactive UIs. React is Declarative, Component-Based and Write Once Use Anywhere.
