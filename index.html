So, what exactly is Matrix?

Matrix is an open-source project that publishes the “Matrix open standard” for secure, decentralised, real-time communication, and its Apache licensed reference implementations. Matrix gives you simple HTTP APIs and SDKs (iOS, Android, Web) to create chatrooms, direct chats and chat bots, complete with end-to-end encryption, file transfer, synchronised conversation history, formatted messages, read receipts and more.
